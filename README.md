# rlxc
Simple side-project to implement a Rust binary using
[LXC](https://github.com/lxc/lxc) to run containers.
Currently covers:

- `lxc-attach` --> `rlxc exec`
* `lxc-start` -> `rlxc start`
- `lxc-stop` -> `rlxc stop`
- `lxc-ls` -> `rlxc list`

as well ass:

- `rlxc help`
- `rlxc version`

# LXC
For information about LXC see [here](https://github.com/lxc/lxc).
